# Personal Portfolio Platform

A self-hosted portfolio platform with dynamic projects, notes, media uploads, admin-managed content, and active feature development.

## Features

- Dynamic project and note management
- Admin dashboard for content updates
- Project galleries and cover images
- PDF/HTML note attachments
- Markdown and math rendering
- Contact form workflow
- Self-hosted production deployment
